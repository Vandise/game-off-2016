## Setup
Clone this repository

```
$> git clone git@github.com:Vandise/game-off-2016.git
```

Install dependencies

```
$> cd game-off-2016 && npm install
```

Start up the game

```
$> npm run start:dev
```

Navigate your web browser to:

```
127.0.0.1:8000
```
